## 📖 About The Project

This repository contains the source code for my **personal portfolio website**, created to professionally showcase my:

- AI & Data Science skill set  
- Academic background and career goals  
- Projects demonstrating practical problem-solving  
- Certifications and continuous learning  
- Internship and professional experience  
- Resume and contact information  

The portfolio is designed to be **clean, elegant, and recruiter-friendly**, ensuring visitors can quickly understand my technical capabilities and interests.

## 🛠 Tech Stack

- **HTML5** – Website structure  
- **CSS3** – Styling and responsive layout  
- **JavaScript** – Interactivity and behavior  
- **Git & GitHub** – Version control  
- **GitHub Pages** – Hosting  

---

### 📌 Structure Explanation

- **index.html** – Core webpage with all portfolio sections  
- **style.css** – Controls colors, fonts, layout, and responsiveness  
- **script.js** – Adds interactive features and smooth behavior  
- **assets/images/** – Stores images used in the website  
- **assets/resume/** – Contains the resume file  
- **README.md** – Documentation for GitHub visitors
